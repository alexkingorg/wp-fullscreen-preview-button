== Description ==

This plugin adds a "preview" button to the distraction-free editor.

== Installation ==

1. Install through the WordPress admin.

_or_

1. Upload `fullscreen-preview-button.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
